# abrirPlanificacion() -> Detalle y Prototipado

##  Propósito
Permitir al Miembro (o Integrante) seleccionar o iniciar la creación de una Tarea para acceder a las opciones de configuración y llevar el sistema al estado `PLANIFICACION_ABIERTO`. Esta transición se inicia desde el estado `SISTEMA_DISPONIBLE`.

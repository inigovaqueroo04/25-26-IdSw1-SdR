## 📅 Contexto
Reunión con el cliente para la definición de requisitos de un *software de gestión de problemas y tareas familiares*.  
El cliente busca una herramienta que permita *organizar, asignar y optimizar tareas* dentro de un entorno familiar con *roles, jerarquías y niveles de recursividad* ilimitados.

---

## 🎯 Objetivo del Sistema
Desarrollar una aplicación web que permita gestionar las tareas del hogar mediante:
- Creación y organización de tareas y subtareas (con recursividad infinita).
- Asignación de tareas a diferentes agentes (miembros de la familia).
- Priorización, geolocalización y optimización de rutas.
- Control de permisos y jerarquías entre usuarios (padres, hijos, etc.).
- Detección de conflictos y tareas solapadas.
- Interfaz intuitiva con listas personalizadas y generales.

---

## 🔔 Funcionalidades Clave

- ✅ Creación y edición de tareas con recursividad infinita.  
- 🗺️ Integración con mapas para rutas óptimas.  
- 📆 Calendario de tareas con confirmación de cumplimiento.  
- 🔐 Control de permisos y aprobaciones.  
- 🧭 Detección de conflictos y compatibilidad de tareas.  
- ⏰ Recordatorios y reprogramación flexible.  
- 📋 Listas personalizadas y generales.

---

## 💡 Ejemplo de Uso

1. El *Padre* crea la tarea “Hacer la compra semanal”.
2. Añade subtareas: “Revisar nevera”, “Hacer lista de compra”, “Ir al supermercado”.
3. Asigna “Hacer lista de compra” al *Hijo 1* (requiere aprobación).
4. El sistema ordena las tareas según la ruta más eficiente para las compras.
5. Al completarse, el usuario marca las tareas como hechas y se actualiza el estado global.

---

## 🧠 Conclusión
El cliente busca un software *inteligente, flexible y colaborativo* que gestione tareas familiares con distintos niveles de complejidad y jerarquía.  
El foco principal está en la *recursividad de subtareas, la **optimización de rutas, y un **sistema de permisos y aprobaciones* adecuado para el entorno familiar.
